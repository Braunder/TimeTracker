Time Tracker API

Аутентификация

POST /api/auth/register
Регистрация нового пользователя
POST /api/auth/login
Вход пользователя и получение токена

Проекты

GET /api/projects
Получение списка всех проектов
Требуется JWT токен
POST /api/projects
Создание нового проекта
Требуется JWT токен
GET /api/projects/{id}
Получение информации о конкретном проекте
Требуется JWT токен
PUT /api/projects/{id}
Обновление информации о проекте
Требуется JWT токен
DELETE /api/projects/{id}
Удаление проекта
Требуется JWT токен

Учет времени

GET /api/time-entries
Получение списка записей времени пользователя
Требуется JWT токен
POST /api/time-entries
Создание новой записи времени
Требуется JWT токен
GET /api/time-entries/{id}
Получение информации о конкретной записи времени
Требуется JWT токен
PUT /api/time-entries/{id}
Обновление записи времени
Требуется JWT токен
DELETE /api/time-entries/{id}
Удаление записи времени
Требуется JWT токен

Отчеты

GET /api/reports/project/{project_id}
Получение отчета по проекту за период
Требуется JWT токен и права менеджера
Параметры запроса: start_date, end_date (формат: YYYY-MM-DD)
